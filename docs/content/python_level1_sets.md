# 파이썬 레벨별 상세 코드 세트 (Lv 1: 출력 기초와 이스케이프 문자)

이 문서는 **Level 1**을 마스터하기 위해 사용자가 타이핑해야 할 **20개의 코드 세트**를 정의합니다. 사용자는 이 20개를 모두 완료해야 레벨을 통과합니다.

---

### [Set 1] 기본 문자열 출력
```python
print("Hello, Python!")
print("Welcome to TypingCode.")
print("Let's start our journey.")
```

### [Set 2] 줄바꿈 이스케이프 (\n)
```python
print("First Line\\nSecond Line")
print("Third Line\\nFourth Line")
print("Newline character is powerful.")
```

### [Set 3] 탭 정렬 (\t)
```python
print("ID:\\tuser_01")
print("Rank:\\tGold")
print("Skill:\\tPython")
```

### [Set 4] 구분자 활용 (sep)
```python
print("Apple", "Banana", "Cherry", sep=", ")
print("Red", "Green", "Blue", sep=" | ")
print("Separate items with style.")
```

### [Set 5] 끝 문자 제어 (end)
```python
print("Loading", end="... ")
print("100%", end=" ")
print("[COMPLETE]")
```

### [Set 6] 따옴표 출력 (Escaping Quotes)
```python
print("He said, \\"Python is easy.\\"")
print('It\\'s a great language.')
print("Mastering quotes is essential.")
```

### [Set 7] 백슬래시 출력 (\\\\)
```python
print("Path: C:\\\\Users\\\\Admin")
print("Use double backslash for one.")
print("Windows paths need escaping.")
```

### [Set 8] 멀티라인 따옴표 (Triple Quotes)
```python
msg = """
Coding is 
Thinking and
Typing.
"""
print(msg)
```

### [Set 9] 수평 정렬 대시보드
```python
print("--- [ STATUS ] ---")
print("CPU", "45%", sep=":\\t")
print("RAM", "72%", sep=":\\t")
print("------------------")
```

### [Set 10] 문자열 반복 연산
```python
print("=" * 20)
print("  SYSTEM STARTUP  ")
print("=" * 20)
print("Initializing resources...")
```

### [Set 11] 복합 출력 (sep & end)
```python
print("Processing", "Data", sep="_", end="... ")
print("Done", end="!")
print("\\nLog saved successfully.")
```

### [Set 12] 빈 줄 출력 및 가독성
```python
print("Header Section")
print()
print("Body content starts here.")
print("Footer Section is next.")
```

### [Set 13] 주석과 코드 분리
```python
# Display application info
print("App Name: TypingCode")
print("Version: 1.0.0")
print("Author: Antigravity")
```

### [Set 14] 이스케이프 종합 선물세트
```python
print("Name:\\t'Python'\\nType:\\t\\"Language\\"")
print("Goal:\\tPerfect Typing")
print("Status:\\tLearning...")
```

### [Set 15] 콤마를 이용한 나열
```python
print("Variables", "Loops", "Classes", sep=", ")
print("Functions", "Modules", "Errors", sep=", ")
print("All are Python concepts.")
```

### [Set 16] 특수 문자 출력
```python
print("Progress [#####-----] 50%")
print("Searching for *.* files...")
print("Found: 25 files.")
```

### [Set 17] 긴 문장 나누어 출력
```python
print("Python is a high-level,")
print("interpreted, general-purpose")
print("programming language.")
```

### [Set 18] 변수와 단순 결합 (문자열)
```python
lang = "Python"
print("Learning", lang, "now.")
print(lang, "is very", "flexible.")
```

### [Set 19] 시스템 경로 예제 (Linux/Mac)
```python
print("User Path: /usr/local/bin")
print("System Log: /var/log/syslog")
print("Directory structure is clear.")
```

### [Set 20] [Challenge] 레벨 1 최종 복습
```python
print("#" * 30)
print(" LEVEL 1 COMPLETE ", sep=" ")
print("#" * 30)
print("Skills unlocked: print, sep, end, \\\\n, \\\\t")
print("Proceed to Level 2?")
```
